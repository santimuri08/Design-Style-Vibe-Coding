## ğŸ¨ Design Principles

### Typography System

```css
/* Body Text */
font-family: 'Open Sans', sans-serif;
font-size: 19px;
line-height: 1.75;  /* 33.25px */
max-width: 720px;   /* ~65 chars/line */

/* Headings */
font-family: 'Merriweather', serif;
h1: 2.5rem (40px)
h2: 2rem (32px)
h3: 1.5rem (24px)

/* Why these sizes? */
Research shows 19px is optimal for comfortable
extended reading on screens, and 1.75 line-height
provides proper spacing for comprehension.
```

### Color Palette

```css
/* Warm, Accessible Colors */
--color-text: #333333;        /* Warm dark gray (not pure black) */
--color-background: #FAFAF8;  /* Off-white (not pure white) */
--color-primary: #2C5F8D;     /* Muted blue */
--color-secondary: #6B7F8F;   /* Sage gray */
--color-accent: #A8DADC;      /* Soft cyan */
--color-highlight: #F0EDE3;   /* Warm beige */

/* Accessibility */
Contrast Ratio: 7:1 (WCAG AAA compliant)
Why? Reduces harsh contrast while maintaining
excellent readability for all users.
```

### Spacing System

```css
/* Modular Scale */
--space-xs: 0.5rem;   /* 8px */
--space-sm: 1rem;     /* 16px */
--space-md: 1.5rem;   /* 24px */
--space-lg: 2.5rem;   /* 40px */
--space-xl: 4rem;     /* 64px */

/* Application */
Paragraph spacing: 1.5em
Section spacing: 4rem
Card padding: 2rem
```

### Layout Principles

1. **Single-column reading** - Optimized for comprehension
2. **Grid-based alignment** - Consistent visual rhythm
3. **Generous margins** - Content never touches edges
4. **Mobile-first responsive** - Progressive enhancement
5. **Content over decoration** - Function drives form
