Project Overview

This website explores Humanist Modernism—a warm, readable, human-centered design approach—while demonstrating its principles through authentic implementation.

Design Principles in Action

This website itself demonstrates Humanist Modernism principles:

1. Generous Typography

Body text is set at 19px with 1.75 line-height—larger and more spacious than typical web standards. This prioritizes comfortable reading over space efficiency.

Notice how easy it is to read entire paragraphs without strain.

2. Optimal Line Length

All text blocks are constrained to approximately 720px wide (60-75 characters per line). Research shows this is optimal for reading comprehension.

Wide margins create breathing room and help eyes track line by line.

3. Warm Color Palette

Text uses #333 on #FAFAFA backgrounds—not pure black on white. This reduces harsh contrast while maintaining WCAG AAA accessibility (7:1 ratio).

Accent colors are muted earth tones: sage gray, warm taupe, soft blues.

4. Humanist Typefaces

Open Sans (sans-serif) and Merriweather (serif) were chosen for their humanist characteristics: varied stroke widths, open apertures, and calligraphic origins.

These fonts are more legible than geometric alternatives like Futura.

5. Accessibility First

Skip links for keyboard navigation, semantic HTML, generous touch targets (44×44px minimum), and clear focus states ensure everyone can access content comfortably.

Universal Design principles were central from the start.

AI Collaboration Story

This project was created through collaboration with Claude, an AI assistant. The process demonstrates how AI can translate design principles into functional code while maintaining authentic style implementation.

"The combination of human design judgment and AI technical execution produced a result that neither could achieve alone."

— Project Reflection

The Process

Phase 1: Research (1 hour)
I studied the Humanist Modernism style guide, collected reference examples from Edward Tufte's books, iA Writer, and Medium.com. Understanding the philosophy—that design serves human needs—was essential before writing any code.

Phase 2: Foundation (2 hours)
Working with Claude, I described the typography requirements: 19px body text, 1.75 line-height, 720px max-width. Claude generated CSS with proper spacing scale and humanist font stack. We iterated 3 times to get the warmth right.

Phase 3: Content Structure (2 hours)
I outlined the content hierarchy and Claude helped structure semantic HTML. We focused on scannable layouts with frequent subheadings, generous paragraph spacing, and clear visual hierarchy through size and weight.

Phase 4: Refinement (2 hours)
Testing on mobile devices revealed touch targets were too small. Claude adjusted button padding and navigation links. We refined color contrast to meet WCAG AAA standards while maintaining warmth.

Phase 5: Polish (3 hours)
Final quality assurance included Lighthouse testing (scores 95+), accessibility review, and comparing against authentic Humanist Modernism examples. Claude helped optimize performance and add semantic ARIA labels. We also added JavaScript enhancements for improved interactivity, including scroll progress tracking, smooth scrolling, and an interactive font size adjuster demonstration.

Time Spent

Total: 10 hours

  • Research & style study: 1 hour
  • Foundation & typography: 2 hours
  • Content structure: 2 hours
  • Component refinement: 2 hours
  • Polish & testing: 2 hours
  • Documentation: 1 hour

Technologies Used

  • HTML5 with semantic elements
  • CSS3 (CSS Grid, Flexbox, Custom Properties)
  • Google Fonts (Open Sans, Merriweather)
  • JavaScript for enhanced interactivity (scroll progress, smooth scrolling, interactive demos)
  • Responsive design (mobile-first approach)

JavaScript Features

To enhance user experience, we implemented JavaScript features that improve navigation and demonstrate design principles:

  • Scroll Progress Bar: Visual indicator at the top of the page showing reading progress
  • Back to Top Button: Smooth scroll functionality for quick navigation
  • Interactive Font Size Adjuster: Demonstrates the impact of typography choices in real-time
  • Smooth Anchor Scrolling: Enhanced navigation between page sections
  • Keyboard Shortcuts: Alt+T to jump to top, Escape to clear focus
