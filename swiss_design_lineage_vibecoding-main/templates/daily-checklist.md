---

## Day 1: Research (1 hour)

**Choose Style (20 min)**
- [ ] Read style guides in `/style-guides/beginner/`, `/intermediate/`, or `/advanced/`
- [ ] Pick one that excites you
- [ ] Read entire guide
- [ ] **My style:** _____________

**Find References (20 min)**
- [ ] Find 10-15 authentic examples
- [ ] Save to `/research/references/`
- [ ] Document sources

**Generate Mockup (20 min)**
- [ ] Describe vision to AI
- [ ] Generate HTML mockup
- [ ] Screenshot and save

---

## Day 2: Context (1 hour)

**AI Creates 3 Content Pages:**
- [ ] Timeline page (history of style)
- [ ] Designer profiles (key practitioners)
- [ ] About page (style explanation)
- [ ] Review for accuracy
- [ ] Add authentic images

---

## Day 3: Planning (30 min)

**Define 3 Sprints:**
- [ ] Sprint 1: Foundation (layout, typography, colors)
- [ ] Sprint 2: Components (nav, cards, buttons, footer)
- [ ] Sprint 3: Polish (responsive, final pages, QA)
- [ ] Set success criteria for each

---

## Days 4-6: Build (6 hours, 2hr/sprint)

### Sprint 1: Foundation (2 hours)
- [ ] Describe to AI
- [ ] AI generates code
- [ ] Screenshot desktop/mobile
- [ ] Compare to references
- [ ] Iterate 2-3 times
- [ ] Run Lighthouse
- [ ] Document decisions

### Sprint 2: Components (2 hours)
- [ ] Describe to AI
- [ ] AI generates code
- [ ] Screenshot desktop/mobile
- [ ] Compare to references
- [ ] Iterate 2-3 times
- [ ] Run Lighthouse
- [ ] Document decisions

### Sprint 3: Polish (2 hours)
- [ ] Complete all pages
- [ ] Responsive refinement
- [ ] Screenshot desktop/tablet/mobile
- [ ] Compare to references
- [ ] Iterate 2-3 times
- [ ] Final Lighthouse (all 90+)
- [ ] Document decisions

---

## Day 7: Reflection (1.5 hours)

**Collaboration Story (30 min)**
- [ ] AI writes reflection on partnership
- [ ] Review and refine
- [ ] Add process screenshots

**Final QA (30 min)**
- [ ] Lighthouse 90+ all categories
- [ ] All pages responsive (desktop/tablet/mobile)
- [ ] All links work
- [ ] Check accessibility
- [ ] No console errors

**Deploy & Submit (30 min)**
- [ ] Deploy to hosting platform
- [ ] Update README (style, time, decisions, URL)
- [ ] Final commit
- [ ] Submit live URL + repo link

---

## Submission Checklist

- [ ] Live site deployed
- [ ] Lighthouse 90+ all categories
- [ ] All pages responsive
- [ ] Collaboration story complete
- [ ] README updated with time log
- [ ] `/research/` folder included
- [ ] GitHub repo link submitted

---

**Daily Reminder:** Screenshot everything, compare to references, iterate 2-3x minimum.
